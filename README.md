# chalix-dev-test
 Chalix Developer Test

## 기술 스택

### Frontend
- React
- CSS

### Backend
- Node.js
- Express

## 복제 및 실행 방법

1. 저장소를 클론합니다.
    ```bash
    git clone https://github.com/Rbcream/chalix-dev-test.git
    ```

2. 프론트엔드 설정
    ```bash
    cd frontend
    npm install
    npm run dev
    ```

3. 백엔드 설정
    ```bash
    cd backend
    npm install
    npm run dev
    ```
